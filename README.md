# Act-3-Drawer-Rutas-Nombradas-en-main

Oye, ayúdame con un código en Flutter. Estoy haciendo una app de una farmacia con un estilo futurista y minimalista que me encanta. Quiero que todo siga esta misma línea:

Colores y Fondo: El fondo de la pantalla debe ser un gris tenue (como 0xFF1A1A1D) y para los detalles, iconos y bordes usa un cian neón (0xFF00E5FF). La barra de arriba (AppBar) tiene que ser color azul oscuro (0xFF1D1E33), con el texto centrado y que combine con el resto.

Estructura del Proyecto: Uso rutas nombradas en el main.dart. Mis páginas están guardadas en una carpeta que se llama LasPaginas (ejemplo: alfa.dart, medicamentos.dart).

El Drawer (Menú): Tengo un MiDrawer personalizado. Necesito que el encabezado tenga un degradado pro y que aparezcan mis datos: Nombre de la empresa, Dirección, Teléfono y Correo. El texto de esos datos ponlo con fuente tipo 'monospace' para que parezca una terminal de computadora.

Navegación: Siempre que programes un botón del menú, usa Navigator.pushReplacementNamed para que la navegación sea limpia.

Look de las Páginas: En cada página nueva que me ayudes a hacer, no olvides poner el AppBar centrado y llamar al MiDrawer(). Si hay logos o imágenes, ponles un poquito de brillo (shadow) color cian para que resalten sobre el gris del fondo.
